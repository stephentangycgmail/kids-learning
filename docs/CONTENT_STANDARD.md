# Content Standard

This document defines the standard JSON formats used by the Kids Learning website.

## General Principles
- UTF-8 encoding
- One topic per JSON file
- Stable schema
- Backward compatible whenever possible

## Vocabulary
Required:
- word
- cn
Optional:
- usage
- tenses
- image
- audio

## Dictation
Metadata:
- id
- title
- level
- unit

Sentences:
- full
- cn

## Grammar
Metadata:
- tense
- title

Items:
- sentence
- highlight
- explanation
- cn

## Reading
Metadata:
- title
- level
- topic

## Quiz
- question
- choices
- answer
- explanation

## Rule
Future learning materials generated by ChatGPT or Codex should follow these schemas to minimize code changes.
