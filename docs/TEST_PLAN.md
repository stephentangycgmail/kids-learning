Use this checklist after changes that may affect learning activities. Record failures in `BUG_TRACKER.md`.

- [ ] Sentence loads
- [ ] Play starts correctly
- [ ] Pause

- [ ] Pause then Play resumes from highlighted word
- [ ] Stop then Play starts from beginning
- [ ] Natural completion resets playback
- [ ] Fast mode
- [ ] Learning mode
- [ ] Browser SpeechSynthesis available
- [ ] No console errors
